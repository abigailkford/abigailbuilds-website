# Abigail Builds - Portfolio Website

A modern, minimal portfolio website for a software engineer, architect, and automation consultant.

## 🚀 Features

- **Responsive Design** - Works beautifully on all devices
- **Clean & Minimal** - Professional aesthetic with plenty of breathing room
- **Smooth Animations** - Subtle fade-in effects and hover states
- **SEO Optimized** - Semantic HTML with proper meta tags
- **Fast Performance** - Lightweight, vanilla JavaScript (no frameworks needed)
- **Accessible** - WCAG compliant with keyboard navigation support

## 🚀 Getting Started

1. **Open the website:**
   - Simply open `index.html` in your browser
   - Or use a local development server for best results

2. **Using VS Code Live Server:**
   - Install the "Live Server" extension
   - Right-click `index.html` and select "Open with Live Server"

3. **Customize the content:**
   - Edit `index.html` to update text, links, and content
   - Modify `styles.css` to change colors, fonts, and spacing
   - Update color scheme in CSS variables (`:root` section)

## 🎨 Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --color-primary: #0F5F5C;        /* Deep Teal */
    --color-primary-dark: #0a4441;   /* Darker Teal for hovers */
    --color-secondary: #b3e2da;      /* Tech Mint */
    --color-accent: #d97c32;         /* Apricot Pro */
    --color-background: #FAF9F6;     /* Off-White */
}
```

### Updating Content

All content is in `index.html`. Simply search for the section you want to edit and update the text.

### Adding Your Own Email

Replace `hello@abigailbuilds.com` with your actual email address in:
- Footer links
- Contact CTAs
- Meta tags

### Adding Social Links

Update the LinkedIn link in the footer:

```html
<a href="https://linkedin.com/in/yourprofile" target="_blank">LinkedIn</a>
```
